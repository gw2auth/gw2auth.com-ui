PSA

The official Guild Wars 2 API will be offline around the release of Guild Wars 2: Secrets of the Obscure (SotO).

SotO will be released at {{sotoReleaseDate}}. The API will be offline from shortly prior to until a few days after release.

While the API is offline, the following will not work:

  • Add or update API Tokens
  • Verify your Guild Wars 2 Accounts
  • Login to external websites using GW2Auth

We are seeing issues with the official Guild Wars 2 API at the moment.

Please note that this might impact GW2Auth. There is nothing you or we can do about it.
